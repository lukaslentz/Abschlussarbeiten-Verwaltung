Abschlussarbeiten-Verwaltung

Start und Überblick

Wofür das Tool gedacht ist

Das Abschlussarbeiten-Tool dient zur lokalen Verwaltung betreuter Arbeiten. Es unterstützt Bachelorarbeiten, Masterarbeiten, praktische Studienphasen, IPs und sonstige betreute Projekte.

Die Anwendung läuft vollständig im Browser. Die Daten werden lokal gespeichert und können zusätzlich als JSON-Datei exportiert werden.

Typische Nutzung: Übersicht pflegen, Termine anlegen, Aufgaben verfolgen, Risiken erkennen und den Betreuungsstand im Gantt-Diagramm visualisieren.
02

Abschlussarbeiten-Verwaltung

Projektdateien öffnen

So startest du das Programm

Öffne die Datei index.html im Browser. Empfohlen wird Firefox oder ein aktueller Chromium-basierter Browser.

  1. Projektordner entpacken.
  2. index.html doppelklicken.
  3. Das Tool startet mit der Übersicht.

Die Dateien styles.css und app.js müssen im gleichen Ordner wie index.html liegen.

03

Abschlussarbeiten-Verwaltung

Übersicht

Alle Arbeiten in einer Tabelle

Die Übersicht zeigt alle gefilterten Arbeiten tabellarisch an. Abgeschlossene Arbeiten werden hellgrau dargestellt.

Ein Klick auf eine Tabellenzeile öffnet den Bearbeitungsdialog.

  • Name: Student*in oder Kandidat*in
  • Art: z. B. Bachelorarbeit, Masterarbeit oder IP
  • Status: aktueller Bearbeitungsstand
  • Risiko: grün, gelb oder rot
  • Nächste Aktion: automatisch aus offenen Terminaufgaben übernommen

Einträge filtern

Schnell die relevanten Arbeiten finden

Oberhalb der Übersicht stehen mehrere Filter zur Verfügung. Die Filter wirken kombiniert.

FilterFunktion
SucheDurchsucht Namen, Themen, Unternehmen, Betreuer*innen und Bemerkungen.
AnzeigeZeigt aktive, abgeschlossene, kritische oder bald fällige Arbeiten.
StatusFiltert nach Bearbeitungsstatus.
RisikoFiltert nach grün, gelb oder rot.
ArtFiltert nach Art der Arbeit.
05

Abschlussarbeiten-Verwaltung

Spalten auswählen

Übersicht individuell anpassen

Über den Button Auswahl Einträge können Spalten ein- und ausgeblendet werden.

Die Auswahl wird lokal gespeichert und beim nächsten Start wiederhergestellt.

  1. Button Auswahl Einträge öffnen.
  2. Gewünschte Spalten aktivieren oder deaktivieren.
  3. Mit Übernehmen speichern.
06

Abschlussarbeiten-Verwaltung

Neue Arbeit anlegen

Eintrag erstellen

Mit + Neue Arbeit wird eine neue Arbeit angelegt. Im Reiter Allgemeines werden Stammdaten und Bearbeitungsstand gepflegt.

  • Name
  • Art der Arbeit
  • Studiengang
  • Thema
  • Betreuer*in 1 und 2
  • Start- und Abgabedatum
  • Status und Risiko
07

Abschlussarbeiten-Verwaltung

Reiter Allgemeines

Stammdaten und Status

Der Reiter Allgemeines enthält alle bearbeitbaren Grunddaten einer Arbeit.

Die Buttons Duplizieren und Löschen werden nur in diesem Reiter angezeigt.

Beim Löschen muss zur Sicherheit der Name des Eintrags exakt eingegeben werden.
08

Abschlussarbeiten-Verwaltung

Termine anlegen

Betreuungstermine verwalten

Termine werden über + Termin im Bearbeitungsdialog oder durch Klick in eine Kalenderwoche im Gantt-Diagramm erzeugt.

Termine werden automatisch nach Datum und Uhrzeit sortiert und durchgehend nummeriert.

Beispiel: Wird ein neuer Termin zeitlich vor Termin 2 eingefügt, passt das Tool die Nummerierung automatisch an.

Abschlussarbeiten-Verwaltung

Termin-Reiter

Notizen und Aufgaben

Jeder Termin besitzt einen eigenen Reiter. Dort können Datum, Uhrzeit, Notiz und Aufgaben gepflegt werden.

Der Termin-Reiter enthält drei getrennte Bereiche:

  • Notiz
  • Aufgaben Betreuer*In
  • Aufgaben Student*In

Die beiden Aufgabenbereiche besitzen zusätzlich jeweils eine Checkbox Aufgabe erledigt.

Aufgaben und Aktionen

Automatische Übernahme in die Übersicht

Die Aufgaben aus dem nächsten offenen Termin werden automatisch in die Übersicht übernommen.

  • Aufgaben Student*In erscheinen in der Spalte Nächste Aktion Student*In.
  • Aufgaben Betreuer*In erscheinen in der Spalte Nächste Aktion Betreuer*In.

In der Übersicht werden Aktionen einzeilig angezeigt. Nach längerem Mouseover erscheint der vollständige Text.

Risikologik

Automatische Eskalation

Das Risiko kann manuell gesetzt werden. Die automatische Logik überschreibt manuelle Einstufungen nicht nach unten, sondern eskaliert nur.

BedingungAutomatisches Risiko
Weniger als 10 Tage bis Termin und offene Aufgabengelb
Weniger als 3 Tage bis Termin und offene Aufgabenrot
Beide Aufgaben erledigtNachfrage, ob Risiko auf grün gesetzt werden soll

Abschlussarbeiten-Verwaltung

Gantt-Diagramm

Zeitliche Planung visualisieren

Das Gantt-Diagramm zeigt die Laufzeit der Arbeiten als Balken und Termine als orange Marker.

  • Klick auf eine Arbeit öffnet den Eintrag.
  • Klick auf einen Terminmarker öffnet den Termin.
  • Klick auf eine leere Kalenderwoche erzeugt einen neuen Termin.
  • Mehrere Termine pro Woche werden als mehrere Marker angezeigt.

Nach 1 Sekunde Mouseover über einem Terminmarker erscheinen Termininformationen.

Abschlussarbeiten-Verwaltung

ICS-Export

Termine in Kalender übernehmen

Jeder Termin kann als ICS-Datei exportiert werden. Die Datei kann anschließend in Kalenderprogramme importiert werden.

Exportiert werden:

  • Titel
  • Datum
  • Uhrzeit
  • Notiz
  • Aufgaben Betreuer*In
  • Aufgaben Student*In

Abschlussarbeiten-Verwaltung

JSON Import und Export

Datensicherung

Im Reiter Daten kann über den JSON Export der vollständige Datenbestand gesichert werden.

Beim JSON-Import werden vorhandene Einträge nicht gelöscht. Importierte Einträge werden ergänzt. Falls IDs bereits existieren, erzeugt das Tool neue IDs.

Empfehlung: Regelmäßig eine JSON-Sicherung erstellen.

Abschlussarbeiten-Verwaltung

CSV Export

Daten weiterverarbeiten

Der CSV Export erzeugt eine tabellarische Datei für Excel oder andere Tabellenprogramme.

Der CSV Export ist besonders nützlich für Berichte, Archivierung oder zusätzliche Auswertungen.

17

Abschlussarbeiten-Verwaltung

Lokale Speicherung

Daten im Browser

Die Daten werden lokal im Browser gespeichert. Zusätzlich kann jederzeit manuell gespeichert werden.

  • Automatisches Speichern nach Änderungen
  • Regelmäßige Speicherprüfung
  • Speichern beim Schließen
  • Manuelles Speichern über den Button

Die Speicherung erfolgt im lokalen Browserprofil.
